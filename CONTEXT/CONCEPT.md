# 🌿 RepoMon 11대 가문 도감 (Updated Final Concept)

## 1. ☕ 커피 가문 (The Coffee Family)

- **매핑 언어:** Java, Kotlin, JVM
- **비주얼 컨셉:** 동글동글하고 윤기 나는 짙은 초록 잎사귀 사이에, 보석 같은 빨간 커피 체리가 콕콕 박혀 있는 귀여운 나무.
- **상징:** "개발자에게 카페인이 필수이듯, 시스템을 지탱하는 지칠 줄 모르는 에너지를 상징합니다."

## 2. 🌻 은행 가문 (The Ginkgo Family)

- **매핑 언어:** JavaScript, TypeScript, Web
- **비주얼 컨셉:** 샛노란 부채꼴 잎사귀가 풍성하게 모여 황금빛 구름처럼 보이는 나무. (JS 로고의 노란색 반영)
- **상징:** "도시 어디서나 잘 자라는 은행나무처럼, 웹 생태계 어디에나 존재하는 범용성을 상징합니다."

## 3. 🐍 뱀식물 가문 (The Snake Plant Family)

- **매핑 언어:** Python, AI, Data
- **비주얼 컨셉:** 땅에서 수직으로 솟아오른 검(Sword) 모양의 잎. 초록색 몸체에 노란 테두리가 있어 세련되고 깔끔함.
- **상징:** "파이썬(뱀)이라는 이름처럼, 유연하고 곧게 뻗어나가는 성장력을 상징합니다."

## 4. 🌲 전나무 가문 (The Fir Family)

- **매핑 언어:** C, C++, Rust, System
- **비주얼 컨셉:** 완벽한 삼각형(크리스마스 트리) 형태. 잎이 바늘처럼 뾰족하고 색감이 짙어 단단하고 차가운 느낌.
- **상징:** "한겨울에도 푸름을 잃지 않는 상록수처럼, 어떤 환경에서도 변치 않는 성능과 최적화를 상징합니다."

## 5. 🌸 벚꽃 가문 (The Blossom Family)

- **매핑 언어:** Swift, Objective-C, iOS (Native Mobile)
- **비주얼 컨셉:** 잎보다 꽃이 더 많은 분홍색 솜사탕 같은 나무. 몽글몽글하고 부드러운 파스텔 톤.
- **상징:** "사용자의 눈을 즐겁게 하는 UI/UX처럼, 가장 화려하고 아름다운 네이티브 앱의 미학을 상징합니다."

## 6. 🎋 대나무 가문 (The Bamboo Family)

- **매핑 언어:** Go (Golang), Node.js
- **비주얼 컨셉:** 시원한 직선으로 뻗은 연두색 줄기. 마디가 뚜렷하고 잎이 날렵함. 군더더기 없는 심플함.
- **상징:** "하루에 수십 cm씩 자라는 대나무처럼, 폭발적인 개발 속도와 간결한 문법을 상징합니다."

## 7. 🌳 참나무 가문 (The Oak Family)

- **매핑 언어:** C#, General, Others (Default)
- **비주얼 컨셉:** 우리가 '나무' 하면 떠올리는 가장 표준적인 모습. 굵은 갈색 기둥과 둥근 초록색 잎 덩어리.
- **상징:** "숲의 중심을 잡는 거목처럼, 분류되지 않은 수많은 언어와 프로젝트까지 모두 포용하여 지탱하는 든든한 기반을 상징합니다."

## 8. 🍁 단풍 가문 (The Maple Family)

- **매핑 언어:** Ruby, HTML
- **비주얼 컨셉:** 주황에서 빨강으로 이어지는 강렬한 그라데이션. 별 모양의 잎사귀가 예술적인 느낌을 줌.
- **상징:** "보석 루비(Ruby)처럼, 코드를 예술의 경지로 끌어올리는 개발자의 감성을 상징합니다."

## 9. 🌵 선인장 가문 (The Cactus Family)

- **매핑 언어:** Shell, Config, DevOps
- **비주얼 컨셉:** 동그란 공 모양(Golden Barrel). 가시가 노란 점처럼 귀엽게 박혀 있고, 작지만 단단해 보임.
- **상징:** "물(커밋)이 부족한 척박한 환경(터미널)에서도 묵묵히 시스템을 지키는 생명력을 상징합니다."

## 10. ✂️ 소나무 가문 (The Artistic Pine Family)

- **매핑 언어:** Assembly, Embedded
- **비주얼 컨셉:** 인위적으로 휜 기둥과 구름처럼 다듬어진 잎. 잘 가꿔진 분재나 조경수 느낌.
- **상징:** "제한된 리소스 안에서 극한의 효율을 깎아내는 장인 정신과 절제미를 상징합니다."

## 🫐 11. 블루베리 가문 (The Blueberry Family) [Team Identity]

- **매핑 언어:** **Flutter, Dart**
- **비주얼 컨셉:** 잎사귀는 작고 동글동글하며, 짙은 남색(Navy)과 보라색(Violet)이 감도는 통통한 열매가 포도송이처럼 뭉쳐 있는 풍성한 관목(Bush).
- **상징:** "작은 열매가 모여 강력한 슈퍼푸드가 되듯, '블루베리 팀'의 핵심 기술인 Flutter를 통해 건강하고 강력하게 성장하는 잠재력을 상징합니다."
